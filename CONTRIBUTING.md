# Contributing to Modular Framework

First off, thank you for taking the time to contribute! It is people like you who make this framework better for everyone.

## 🌿 Branching Strategy

To keep our production code stable, we follow a strict branching model:

1.  **main**: This branch contains the latest stable, published version of the framework. **Direct PRs to main will be closed.**
2.  **dev**: This is our integration branch. **All Pull Requests must be targeted at the `dev` branch.**

## 🚀 How to Contribute
